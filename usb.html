USB(Universal Serial Bus)

WHY USBs WERE DEVELOPED?

  • The goal of developement of USB was to make the connection between devices and computer much easier.
  • This is done by:-
    1. Making one standard connector.
    2. Making connected devices easier to configure.
    3. Provide fast transfer rates.

BENEFITS OF USBs

  • USB port is hot swappable which means you don't need to turn off the computer in order to disconnect it with a device, you can just unplug the USB.
  • USB self-configures itself which means that it eliminates the need for extensive configuration or any configuration at all.

    • There are many more benefits.

USB VERSIONS OVER THE YEARS:-

  1. USB 1.0 :
    • Release date : 1996
    • Speed : 1.5Mbit/s
  2. USB 1.1 :
    • Release date : 1998
    • Speed : 12 Mbit/s
  3. USB 2.0 :
    • Release date : 2000
    • Speed : 480 Mbit/s
  4. USB 3.0 :
    • Release date : 2008
    • Speed : 5 Gbit/s
  5. USB 3.1 :
    • Release date : 2013
    • Speed : 10 Gbit/s
  6. USB 3.2
    • Release date : 2017
    • Speed : 20 Gbit/s
  7. USB 4
    • Release date : 2019
    • Speed : 40 Gbit/s

USB CABLES AND CONNECTORS :

  • In addition to different speeds of USB, there are also different types of cables, connectors and ports.
  • USB cables will typically have two different types of connectors on each end of the cable.
  • Now one end is always same i.e. Type A while the other end can be Type A Type B or Type C which means that Type A connector is the host connector.

Depending on what we learned above below are the different types of cable :

  1. TypeA-TypeB
  2. TypeA-Type miniB
  3. TypeA-Type microB
  4. TypeA-Type B 3.0
  5. TypeA-Type microB 3.0
  6. TypeA-TypeC
  7. TypeC-TypeC

COLOR BASED ON USB VERSION :
