DNS (Domain Name Server)

  • In the world of networking, computers don't go by names like humans do , they go by numbers.
  • The reason they go by numbers is because computers only and only understand numbers , so they will be able to talk to each other and identify each other via numbers.
  • These numbers are called ip addresses
  • Now... In order for it to be possible to establish communication between humans and computers while also making the communication easier, networking engineers developed DNS.
  • DNS stands for Domain Name Server and it gives names to numbers or more specifically it gives domain names to ip addresses
  • So if you type in a web address in your web browser, DNS will resolve the name to a number, because the only thing a computer understands are numbers.
